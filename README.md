Hangi Yalan Dünya Karakterisin? (Streamlit App)

Bu proje, Python ve Streamlit kütüphanesi kullanılarak geliştirilmiş etkileşimli bir kişilik testidir. Kullanıcıların verdiği cevaplara göre arka planda çalışan ağırlıklı puanlama algoritması, kullanıcının ruh ikizi olan "Yalan Dünya" karakterini tespit eder.
